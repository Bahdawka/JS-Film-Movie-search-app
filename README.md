# JS Film Movie Search App

A simple web application for searching movies using the [OMDb API](https://www.omdbapi.com/). Users can search for movies by name, view movie posters, and choose whether to add new search results to the existing list or replace them.

## Features

- Search for movies by title (minimum 4 characters).
- Displays movie posters and titles.
- Option to append search results to the current list or replace the list.
- Responsive and clean UI.

## Usage

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Enter a movie name (at least 4 characters) in the search field.
4. (Optional) Check "Add movies to existed list" to append results instead of replacing the list.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- [OMDb API](https://www.omdbapi.com/)
